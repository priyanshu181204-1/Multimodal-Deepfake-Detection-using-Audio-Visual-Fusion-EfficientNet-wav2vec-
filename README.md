# Multimodal Deepfake Detection (Audio + Visual)

##  Overview
This project presents a deepfake detection system that combines visual and audio information to improve detection accuracy in real-world conditions.

##  Key Features
- Multimodal learning (Audio + Video)
- EfficientNet-B4 for visual feature extraction
- wav2vec 2.0 for audio processing
- Cross-modal attention fusion
- Robust to compression and noise
- Explainability using Grad-CAM

##  Model Architecture
- Visual Stream → EfficientNet-B4
- Audio Stream → wav2vec 2.0
- Fusion → Cross-modal attention
- Classifier → MLP

##  Datasets Used
- FaceForensics++
- DFDC (Deepfake Detection Challenge)
- Indian multilingual dataset

##  Technologies
- Python
- PyTorch
- Deep Learning
- Computer Vision
- Audio Processing
